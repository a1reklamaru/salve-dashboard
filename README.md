# Сальве Dashboard

Компактный статический дашборд рекламных показателей по месяцам для двух клиентов: Сальве Кидс и Сальве Взрослая.

## Что внутри

- переключение между клиентами сверху страницы;
- выбор и сравнение месяцев;
- KPI по показам, кликам, CTR, расходу, конверсиям и CPA;
- графики для сравнения 2+ месяцев;
- кампании сгруппированы по месяцам и раскрываются отдельно;
- мобильная версия с компактными KPI и карточным видом кампаний.
